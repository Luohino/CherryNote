# CherryNote - Quick Start Guide

## - Compilation Complete!

Your CherryNote application has been compiled successfully.

**Executable:** `CherryNote.exe` (157 KB)
**Status:** Ready to run

---

##  How to Run

Simply double-click `CherryNote.exe` or run from command line:
```
CherryNote.exe
```

The application will:
1. Launch a small floating window (280×420 px)
2. Stay on top of all other windows
3. Load any previously saved notes
4. Be ready for input

---

##  Usage

### Adding Tasks
Press **Ctrl + Enter** to add a new numbered task:
```
1. First task
2. Second task
3. Third task
```

### Marking Completed Tasks
Select a task and press **Ctrl + D** to toggle completion:
```
1. Buy milk ✔
2. Call mom
3. Write code ✔
```

### Moving the Window
Click and drag the **"CherryNote"** title bar to move the floating window anywhere on your screen.

### Editing
- Use arrow keys to navigate
- Backspace/Delete to remove text
- Enter for manual line breaks
- Auto-save triggers 1 second after you stop typing

---

##  Data Storage

Your notes are automatically saved to:
```
CherryNote.txt
```

Located in the same directory as `CherryNote.exe`

This file is a plain text file that you can:
- Edit manually in any text editor
- Backup to cloud storage
- Share with others
- Open on any system

## If Something Goes Wrong

**Application won't start**
- Verify Windows 7 or later
- Try running as Administrator
- Check antivirus isn't blocking it

**Notes not saving**
- Ensure folder is writable
- Check `CherryNote.txt` permissions
- Make sure disk has space

**Window looks strange**
- Try restarting the application
- Check Windows display scaling settings

**Memory usage high**
- Close the application and reopen
- Check if another instance is running
